# tesuto

A Vite, React, and TypeScript starter app for beginning development quickly.

## Requirements

- Node.js 24 or newer
- npm 11 or newer

## Development

Install dependencies:

```sh
npm install
```

Start the local development server:

```sh
npm run dev
```

Run checks before committing:

```sh
npm run lint
npm run build
```

## Project structure

- `src/App.tsx` - main application UI and task state
- `src/App.css` - application layout and component styles
- `src/index.css` - global theme and reset styles
- `vite.config.ts` - Vite configuration
